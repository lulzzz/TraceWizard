Trace Wizard Command Line Help

You can run Trace Wizard from the Windows Command Line in order to automate certain functions.

Standard wildcard characters * and ? are supported.

Examples

> c:/MyPrograms/TraceWizard.exe c:/MyData/mylog.mdb c:/MyData/mytrace.twdb
Opens log mylog.mdb and trace mytrace.twdb in Trace Wizard.
> c:/MyPrograms/TraceWizard.exe c:/MyData/*.csv
Opens log files with csv file extension in Trace Wizard.
> c:/MyPrograms/TraceWizard.exe /mdbtocsv c:/MyData/*.mdb
Exports log files with mdb file extension to log files in csv log format. For example, for the log file mylog.mdb, the log file mylog.csv will be generated (and overwritten if previously existing). Runs unattended, can be script-driven.
> c:/MyPrograms/TraceWizard.exe /tdbtotwdb c:/MyData/*.tdb
Exports TW4 traces with tdb file extension to TW5 traces in twdb trace format. For example, for the TW4 trace file mytrace.tdb, the TW5 trace file mytrace.twdb will be generated (and overwritten if previously existing). Runs unattended, can be script-driven.
> c:/MyPrograms/TraceWizard.exe /logtotwdb c:/MyData/mylog.mdb c:/MyData/mylog2.csv
Exports log files to TW5 traces without machine classification. For example, for the log files mylog.mdb and mylog2.csv, the TW5 trace files mylog.twdb and mylog2.twdb will be generated (and overwritten if previously existing). Runs unattended, can be script-driven.
