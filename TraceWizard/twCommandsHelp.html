Trace Wizard Commands Help

You can use Trace Wizard Commands to select events that meet criteria you specify in the Command Panel.

A Trace Wizard Command has three parts: Action [Fixtures] [Constraints]. You can see examples here.

Action

Specify the Action you want to perform.

select next
select n
sel next
sel n
s next
s n
next
n 		Selects Next event that meets the criteria.

p 		Selects Previous event that meets the criteria.

sel all

all

view next

v next
v n 		Displays the next currently selected event that meet the criteria.

view p

classify [similar] as

c [sim] 		Classifies selected events as specified fixture (optionally classifying events that are similar to the selected events).
time
t 		Position graph at specified time.
refresh Refreshes similar counts (perform before executing Similar command)

undo (redo) Undoes previous classification.

go to end

Fixtures

You can optionally include or exclude Fixtures as part of the action criteria.

The Fixtures you list must be either all included or all excluded. You cannot include some yet exclude others (for example: leak -shower).

Specifying Fixtures is Optional. You can direct Trace Wizard to consider all Fixtures by not listing any Fixtures in your Command.

leak shower faucet humidifier
l s f humidifier
l s f m 		Includes only events that are Leak, Shower, Faucet, or Humidifier (while excluding all others).
-shower -irrigation
-s -i 		Excludes events that are Shower or Irrigation (while including all others).

Constraints

You can optionally include multiple attribute constraints as part of the action criteria.

volume < 3.0
v < 3.0 		Includes only events whose volume is less than 3.0.
peak > 2 Includes only events whose peak is greater than 2.
duration > 2:00 Includes only events whose duration is greater than 2 minutes.
volume < 3.0 peak > 2
vol < 3.0 pk > 2 		Includes events whose volume is less than 3.0 and whose peak is greater than 2.
1st Includes only events designated as 1st cycle.
-1st Excludes events designated as 1st cycle.
similar
sim 		Includes only events that are similar to currently selected event.
super Includes only super (simultaneous) events.
manual Includes only manually classified events.
list Includes only events classify using fixture list.
machine Includes only events classify using machine learning.
approved Includes only events whose classification is approved by user.
toilet1 Includes only events that satisfy first toilet in fixture list.
notes Includes only events with user notes.
