Trace Wizard Installation

System Requirements

Supported Operating Systems

  • Windows XP Service Pack 2 or above
  • Windows Vista
  • Windows 7

Hardware Requirements

  • Architecture: x86 or x64
  • Minimum: Undetermined
  • Recommended: 2.1 GHz or higher CPU, 4.00 GB RAM, 1280x800 display or above

Software Requirements

  • .NET Framework, version 3.5 SP1
  • Office 2007 System Driver (not the Access 2007 Runtime; required for MDB log format; optional if using other log formats)
  • A Speech Recognition Engine (optional)

Web Services

Internet access is required for online help and version checking and updating (optional)

Installation Troubleshooting

  • To install Trace Wizard, simply download and unzip the application to a folder of your local drive. There is no separate installation program.
  • If Trace Wizard will not come up at all, please review System Requirements above. The .NET Framework might not be installed.
  • If Trace Wizard will not load MDB or TDB files, please review System Requirements above. The Office 2007 System Driver might not be installed. If you have installed Access 2010 (or Office 2010) or later, please reinstall the Office 2007 System Driver.

.NET Framework

To install the .NET Framework, administrative access to your computer may be required. This software is pre-installed on some systems. To confirm whether this software is already installed, use a registry viewer (such as regedt32.exe) and confirm the existence of an entry HKLM\Software\Microsoft\NET Framework Setup\NDP\v3.5\Install. Verify that the value name Install has value data = 1; and that the value name SP has value data = 1.

Office 2007 System Driver

To install the Office 2007 System Driver, administrative access to your computer may be required. This software is pre-installed on some systems. To confirm whether this software is already installed, use a registry viewer (such as regedt32.exe) and confirm the existence of a registry entry HKEY_LOCAL_MACHINE\SOFTWARE\Classes\Microsoft.Jet.OLEDB.4.0 and subkey CLSID with value Default = {dee35070-506b-11cf-b1aa-00aa00b8de95}. Note: It is not necessary to have Microsoft Office 2007 installed, only this driver. This is not the driver version installed by Trace Wizard version 4.

Speech Recognition

To enable speech recognition in Trace Wizard (for classifying selected events):

  • A microphone is required, preferably a headset one.
  • Edit the *.config file in the TW directory. Set IsSpeechRecognitionEnabled value to True. Restart Trace Wizard.
  • If a speech recognition engine is not already installed, Windows may try to install one when Trace Wizard first needs it.
  • Speech Recognition is built into Windows Vista and Windows 7. See Control Panel (Speech Recognition).
  • It is recommended that users 'train' the speech recognition engine to their voice and environment. See Control Panel.
