5.2.2.0 Thu, May 23, 2013

Don't require unique file extensions for logger format; peek inside to determine format. Support configurable DateTimeFormat and NutationRate settings in .config files. For each setting, TW will first look in the .config file with the same filename as the logger file (e.g., L-07596S011_pilot home 1.config). If the setting is not found there, it will then look for it in logger.config in the same folder. Here is a sample of the config file contents:

DateTimeFormat,dd/MM/yy HH:mm:ss
NutationRate,0.01388888

Known Bugs

Reported in Bug # Notes
5.1.0.17 127 Commands that use fixture list (e.g., "toilet1") don't use latest edits to list. Workaround:Trigger save of fixture list edits by classifying using Fixture List before executing fixture list item command (after editing fixture list).
5.1.0.9 122 Event context menu shows access keys for add/apply(incorporate) menu items, but pressing those keys (2 and 3) does not execute requested function. Workaround: use mouse or up/down arrows to access the menu item, or access the feature another way (e.g., select event, press 2 or 3).
5.1.0.8 121 When moving fixture row to top of fixture list, the cursor enters the first attribute textbox instead of staying on the row selected. Workaround: Once the row has reached the top, to move it back down, reselect the row's row selector button.
5.1.0.8 120 Crash when undoing the vertical merge of certain super events that can only be created with TW4. Workaround: No plans to fix, only affects TW4 traces.
5.0.95.0 117 Contiguous merge not offered via context (pop-up) menu if target event has super event. Workaround: Perform merge via drag-and-drop.
5.0.90.0 111 If user's Windows date/time format is set to day/month/year, TW crashes when trying to read CSV log format. Workaround: Windows Control Panel: Region and Language Settings: do not use day/month/year setting. Instead, use month/day/year (traditional in the US) or year/month/day.
5.0.58.6 106 Vertical scroll not functional in Compare Analyses mode.
5.0.44.0 94 Resizing TW window scrolls graph to left end (beginning).
